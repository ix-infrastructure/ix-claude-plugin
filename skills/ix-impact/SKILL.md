---
name: ix-impact
description: Analyze the blast radius and downstream impact of changing a symbol, file, or module using Ix Memory
argument-hint: <symbol or file>
---

Run `ix impact $ARGUMENTS --format json` using the Bash tool.

Present the results as a clear breakdown of: what depends on the target, what would be affected by a change, and any high-risk or widely-used dependencies to be aware of before making edits.
